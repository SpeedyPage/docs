---
title: Hosting resource limits
description: Memory, CPU, entry process, and I/O limits for SpeedyPage Web Hosting, WordPress Hosting, and Reseller Hosting plans.
---

SpeedyPage uses LVE (Lightweight Virtual Environment) to manage resources per cPanel account. These limits define the maximum CPU, memory, and processes your account can use on the server.

## Web Hosting

| | Starter | Premium | Enterprise |
| --- | --- | --- | --- |
| Memory | 2 GB | 3 GB | 4 GB |
| CPU cores | 1 | 1 | 2 |
| Entry processes | 25 | 50 | 100 |
| I/O and IOPS | Unlimited | Unlimited | Unlimited |

## WordPress Hosting

| | Starter | Premium | Enterprise |
| --- | --- | --- | --- |
| Memory | 2 GB | 3 GB | 5 GB |
| CPU cores | 1 | 2 | 3 |
| Entry processes | 25 | 50 | 100 |
| I/O and IOPS | Unlimited | Unlimited | Unlimited |

## Reseller Hosting

All Reseller Hosting plans share the same per-account resource limits:

| | All plans |
| --- | --- |
| Memory | 2 GB |
| CPU cores | 1 |
| Entry processes | 25 |
| I/O and IOPS | Unlimited |

<Tip>
If you're consistently hitting resource limits, check your cPanel **Resource Usage** page to identify which limits are being reached. You can upgrade your plan or optimize your site to reduce resource consumption.
</Tip>
