Softaculous is an auto-installer included with all SpeedyPage Web Hosting, WordPress Hosting, and Reseller Hosting plans. It provides one-click installation for over 500 applications, including content management systems, forums, e-commerce platforms, and more.

## Install an application

This example walks through installing WordPress, but the process is similar for any application in Softaculous.

<Steps>
  <Step title="Open Softaculous">
    Log in to cPanel through your [client area](https://my.speedypage.com) or by visiting `yourdomain.com/cpanel`. Go to the **Software** section and click **Softaculous Apps Installer**.
  </Step>
  <Step title="Select the application">
    Find WordPress in the featured applications or use the search bar. Click the application name, then click **Install Now**.
  </Step>
  <Step title="Configure installation settings">
    Set the following options:

    - **Choose Protocol** — select `https://` if you have an SSL certificate (recommended)
    - **Choose Domain** — select the domain to install on
    - **In Directory** — leave blank to install on the root domain, or enter a subdirectory name (e.g., `blog`)
    - **Site Name** and **Site Description** — enter your site details
  </Step>
  <Step title="Set admin credentials">
    Under **Admin Account**, set:

    - **Admin Username** — choose a username (avoid `admin` for security)
    - **Admin Password** — use a strong password
    - **Admin Email** — your email address for password resets and notifications
  </Step>
  <Step title="Install">
    Review your settings and click **Install**. Softaculous will set up the application and display a confirmation with your site URL and admin login link.
  </Step>
</Steps>

## Manage installed applications

To view, update, or remove installed applications:

1. In Softaculous, click the **Installations** icon (the blue square) at the top of the page
2. You'll see a list of all installed applications with options to **Edit**, **Clone**, **Create Staging**, or **Remove**

<Tip>
Softaculous can send you email notifications when application updates are available. Enable this in the Softaculous settings to keep your software up to date.
</Tip>
