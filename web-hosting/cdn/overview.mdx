SpeedyPage CDN is powered by Bunny.net and is included on selected hosting plans. It caches and delivers your site's content from a global network of edge servers, reducing load times for visitors regardless of their location.

## Network

| Specification | Detail |
| --- | --- |
| Points of presence (PoPs) | 119+ locations worldwide |
| Network capacity | 200 Tbps+ |
| Average global latency | Sub-25ms |
| Routing engine | SmartEdge with real-time optimization |

SmartEdge automatically routes each request to the fastest available PoP based on real-time network conditions, rather than relying solely on geographic proximity.

## Features

### Content caching

Static assets (images, CSS, JavaScript, fonts, and other files) are cached at edge servers around the world. When a visitor requests your site, content is served from the nearest PoP instead of your origin server.

### Bunny Shield

An integrated security layer that includes an AI-powered WAF, DDoS protection, rate limiting, and bot mitigation. See [Bunny Shield](/web-hosting/cdn/bunny-shield) for details.

### Image optimization

Automatic image compression, format conversion to WebP and AVIF, and lazy loading — all without code changes. See [image optimization](/web-hosting/cdn/image-optimization) for details.

### Compression

Brotli and Gzip compression are applied automatically to text-based content (HTML, CSS, JavaScript, JSON, XML), reducing transfer sizes.

## Availability

SpeedyPage CDN is included on selected Web Hosting, WordPress Hosting, and Reseller Hosting plans. It can also be added to any hosting plan as an add-on through your [client area](https://my.speedypage.com).
