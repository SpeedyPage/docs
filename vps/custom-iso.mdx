If you need an operating system that isn't in our [standard list](/vps/operating-systems), you have two options: netboot.xyz (self-service) or a custom ISO upload (via support).

## Netboot.xyz

SpeedyPage includes [netboot.xyz](https://netboot.xyz), an open-source network boot tool that lets you install a wide range of operating systems — including Linux distributions, FreeBSD, and Windows — without uploading an ISO file. You can boot into netboot.xyz directly from the VirtFusion control panel.

## Custom ISO

For anything not available through netboot.xyz:

1. Find a direct download URL for your ISO file
2. Open a support ticket through your [client area](https://my.speedypage.com) with the download URL
3. The support team will review and attach the ISO to your VPS
4. Install the OS through the VirtFusion console as normal
