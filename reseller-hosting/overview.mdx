SpeedyPage Reseller Hosting gives you everything you need to run your own hosting business. Every plan is fully white-label — your clients interact with your brand, not SpeedyPage.

## What's included

All Reseller Hosting plans include:

- **WHM (Web Host Manager)** — create and manage client cPanel accounts, set resource limits, and configure server-level settings
- **cPanel for clients** — each client gets their own cPanel to manage their website, email, and files
- **LiteSpeed Web Server** — high-performance web server with built-in caching
- **NVMe SSD storage** — fast storage for all accounts
- **Imunify360** — server-level security with malware scanning and firewall protection
- **Free SSL certificates** — automatically provisioned for all client domains via AutoSSL
- **MailChannels email delivery** — outbound email routed through MailChannels for improved deliverability
- **Anycast DNS** — low-latency DNS resolution via `ns1.speedy.page` and `ns2.speedy.page`
- **Daily backups** — automatic backups with 31-day retention
- **DDoS protection** — network-level protection included on all plans
